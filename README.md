# Bread Basket Analysis

##  Overview
This project performs Market Basket Analysis on a bread basket
dataset to identify frequent itemsets and association rules.
The analysis helps understand customer purchasing patterns.

## Dataset
- Contains transaction data from a bread basket
- Each transaction represents items purchased together

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Mlxtend

## Workflow
1. Data loading and cleaning
2. Transaction encoding
3. Frequent itemset generation using Apriori
4. Association rule mining
5. Result interpretation

## Key Insights
- Identified commonly purchased item combinations
- Generated association rules with support, confidence, and lift

## Use Case
This analysis can help retailers:
- Improve product placement
- Design better promotions
